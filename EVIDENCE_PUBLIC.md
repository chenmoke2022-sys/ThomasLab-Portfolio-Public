# 证据说明（脱敏版）| Evidence (Desensitized)

> **原则**：我公开"证据入口与结构"，不公开"可直接复刻的核心实现与参数"。  
> **目标**：让评审方快速判断"是否值得进入 30 分钟技术评审"。

---

## 一、我如何做证据 | How I Do Evidence

### 我的证据体系（三层）
1. **Claim Registry（单一事实来源）**：所有数字口径统一管理，避免口水仗
2. **Evidence Suite（一键复现）**：关键主张可通过命令行复现并输出审计工件
3. **CI Guarding（回归守护）**：持续集成确保主张不漂移

### 证据级别分类
- **Verified（已验证）**：有完整实验、数据、图表、复现入口
- **Planned（计划中）**：有设计方案但尚未完整实验
- **Idea（想法）**：仅有概念，无实验支持

> **我只对 Verified 级别的主张负责。**

## 二、可公开的证据（脱敏版）| Public Evidence

### 1. 门控延迟分布（System 0/0.5 Gating Latency）
**主张**：消费级 CPU 上，门控（normalize → 共振检查 → 路由）可在个位数毫秒内完成。

**脱敏数据**：
- 门控开销：**个位数毫秒级**；尾部为**十毫秒级**
- 分布口径：分位数（P50/P95/P99）与完整直方图在私有证据包（线上评审可现场展示）

**测试条件**：
- 固定一组代表性负载与门控策略（具体参数与阈值不公开）
- 对照：将门控开销与慢路径（模型推理/远程调用/人工确认）做量级对比

**如何复核**：技术评审时可屏幕共享完整实验日志与分布图。

---

### 2. 100k 节点图压测（Graph Memory Stress Test）
**主张**：10 万节点图谱压测下，Golden Memory（黄金记忆）始终保持 **Rank=1**，内存增长可控。

**脱敏数据**：
- Golden Memory Rank: **始终 = 1**（在 1k → 100k 节点过程中）
- RAM 增长：**十几 MB 量级**（近似线性）
- 检索延迟：**毫秒级**（完整分布与环境信息在私有证据包）

**如何复核**：技术评审时可演示压测命令与输出报告。

---

### 3. 共振增益分布（Universal Pulse Entrainment Gain）
**主张**：在漂移/噪声/多音调 + 周期重锁相条件下，UPE 控制器相比固定频率基线有**正中位数增益**，但分布有尾部。

**脱敏结论（多次随机试验）**：
- 相比固定频率基线：**中位数为正增益**（存在尾部与极端样本）
- 分布口径、采样方法、参数敏感性曲线：在私有证据包中展示

**如何复核**：技术评审时可展示分布图与参数敏感性分析。

---

### 4. 审计追踪覆盖（Audit Trace Coverage）
**主张**：每次路由决策都有 trace 字段记录：为什么快路径、为什么回退、置信度、预算消耗。

**脱敏示例**（字段结构，非真实数据）：
```json
{
  "trace_id": "req_20251231_001",
  "timestamp": "2025-12-31T10:00:00Z",
  "route_decision": "system_0_fast",
  "confidence": 0.89,
  "budget_consumed": 0.02,
  "fallback_triggered": false,
  "reason": "whitelist_match",
  "rollback_safe": true
}
```

**如何复核**：技术评审时可展示完整 trace 日志（脱敏版）。

## 三、不公开的内容 | What I Don't Publish

为了保护核心 IP 与避免被低成本复刻：
- ❌ 核心路由策略的具体阈值与规则表
- ❌ 关键参数、门控阈值、预算策略细节
- ❌ 可直接跑出核心效果的配置与数据
- ❌ 专利相关的可复刻实现细节

> **这些内容可在进入付费 PoC 或必要保密流程后展开。**

## 四、如何评审 | How to Review

### 我建议的评审方式（30 分钟）
1. **10 分钟**：我讲"问题→机制→边界→交付物"
2. **10 分钟**：你指定你们的一条真实链路/指标
3. **10 分钟**：我给出 4–8 周付费 PoC 的里程碑与验收口径（可回滚）

### 评审后下一步
- **会后 30 分钟**：我发会议纪要（冻结 KPI 与 Guard）
- **会后 24 小时**：我发 PoC 提案 v1（含验收口径、回滚方案、费用节点）
